# starubucks-product-recommendation-system-rag
AI-powered product recommendation system for the UCLA Starbucks Data Challenge. Uses a 3-stage Search/RAG pipeline featuring Vertex AI (Gemini 2.5 Flash) for structured constraint extraction and all-mpnet-base-v2 embeddings for semantic ranking. Achieved an NDCG of 0.965 and Recall of 0.977.
